1. goto blumtod directory
   ```
   cd blumbot
   ```

2. install the require library
   ```
   python -m pip install -r requirements.txt
   ```

3. Edit `data.txt`, input you data token in `data.txt`, find you token in [How to Find Account Token](#how-to-find-account-token). One line for one data account, if you want add you second account add in new line!

4. execute the main program 
   ```
   python bot.py
   ```
